# News API (TypeScript + Express)
A lightweight backend service fetching latest news via GNews.io API.

## Run locally
1. Clone repo
2. Create `.env` with: NEWS_API_KEY=your_key_here
3) npm i
4) npm run dev 

### Endpoints
GET /health
GET /api/top?country=us
GET /api/search?q=bitcoin
